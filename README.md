# FS25AutoDriveCourses
 Couses I Have Made for Farm Sim 2025

 The approach I'm taking is
   - get all the roads in with two way traffic where there is room.
   - Put and entrance point in for every field.
   - Put destinations in for the interactive things on each farm.
   - Put destinations in all the default sell and buy points
   - I'm generally playing while I build these so some fields will be more complete with the following:
     - a TractorWH point that is on the field.
     - a TrailerWH point where automated trailers can sit for unloading into or refilling.
     - a TractorUL point where the tractor can unload into something parked at the above point. 

If it's in the WIP folder I'm still working on may or may not be complete in many ways. There will be a testing folder when I get one to that stage.  Feel free to download any of them in any state.  Please report any problems you encounter.  Some time in testing I will clean up any customizations I did in building my farm that will not be there when a game is fresh.
